# Подсказки по коьфндной строке

Команда смены директории 
```sh
cd c:\folder_name
```

Команда отображения текущей директории. Mc, Lin
```sh
pwd
```

Листинг текущей дирректории
Windows:
```sh
dir
```
Lin, Mc
```sh
ls
```

Удаление файла в Windows: 
```sh
del <filename>
```
в операционке Lin, Mc
```sh
rm <filename>
```
